## PayFlow — Distributed Payment Processing System

PayFlow is a simplified distributed payment infrastructure inspired by systems like Stripe and Razorpay.
It enables merchants to accept payments, track transactions, and receive payment status updates in real time.

The system is designed to demonstrate key distributed system principles such as:

* Idempotent payment processing
* Event-driven architecture
* Eventually consistent workflows
* Ledger-based wallet accounting
* Fault tolerant webhook delivery

The goal of this project is to simulate how real payment infrastructure handles millions of transactions reliably.
